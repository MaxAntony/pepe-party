## Installacion

### Ejecutar directamente

solo escribe el comando

```
npx pepe-party
```

## Instalar y ejecutar

instala globalmente via npm

```
npm i -g pepe-party
```

se habra agregado al `path` y ahora solo ejecuta

```
pepe-party
```

## Salir

Puedes salir en cualquier momento presionando `Ctrl+C`.
